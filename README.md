YouHider is a browser extension to hide YouTube distractions. Hide distractions like Shorts and Playables, or metrics like view count and video duration.

https://github.com/user-attachments/assets/47622270-7409-46cd-a60c-8e82e428e3d6

> [!WARNING]
> YouHider is currently in beta. Some features may not work perfectly yet.

## Features

### Content

- Hide Comments
- Hide End Screen
- Hide Home Feed
- Hide Member Videos
- Hide Mixes
- Hide Playables
- Hide Related Videos
- Hide Shorts

### Metrics

- Hide Comment Count
- Hide Like Count
- Hide Subscriber Count
- Hide Upload Date
- Hide Video Duration
- Hide View Count

## Installation

### <img src="https://upload.wikimedia.org/wikipedia/commons/e/e1/Google_Chrome_icon_%28February_2022%29.svg" width="24" height="24" alt="Chrome" align="absmiddle"> Chrome Web Store

1. Go to the Chrome Web Store page
2. Click "Add to Chrome"

### <img src="https://upload.wikimedia.org/wikipedia/commons/a/a0/Firefox_logo%2C_2019.svg" width="24" height="24" alt="Firefox" align="absmiddle"> Firefox Add-ons

1. Go to the Firefox Add-ons page
2. Click "Add to Firefox"

## Usage

1. Go to [youtube.com](https://www.youtube.com/)
2. Open the YouHider extension in the browser toolbar
3. Toggle checkboxes to hide distractions
